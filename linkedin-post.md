## Post Copy

I was spending 15 hours every week on LinkedIn prospecting.

Searching for leads. Writing connection requests. Tracking conversations in spreadsheets.

At €50/hour, that's €3,000/month down the drain.

So I built a 4-agent system that runs on autopilot:

**Agent 1: Scout**
- Searches LinkedIn using ICP criteria
- Extracts profile data automatically
- Adds qualified leads to queue

**Agent 2: Qualify**
- Scores each lead 1-10 using AI
- Rejects mismatches instantly
- Only passes hot leads forward

**Agent 3: Outreach**
- Writes personalized messages (not generic AI slop)
- References specific profile details
- Sends via LinkedIn API

**Agent 4: Follow-up**
- Tracks who accepted/replied
- Sends 3-touch sequences
- Moves converts to CRM

**The result:**

15 hours/week → 30 minutes/week

I just review what the agents found. Everything else is automated.

**Tech stack:**
- n8n (orchestration)
- Claude 3.7 Sonnet (qualification + messaging)
- Composio (LinkedIn API)
- Notion (CRM)

**Total cost:** €47/month

**ROI:** €3,000 saved - €47 = €2,953/month profit

**The secret nobody talks about:**

Most people use AI to write content.

Smart founders use AI to run their entire revenue engine.

While competitors manually send 10 connection requests a day, my agents send 200 - all personalized, all tracked, all converting.

**Want the blueprint?**

I documented the entire system:
- All 4 agent workflows (n8n JSON)
- Message templates that convert
- Notion CRM setup
- One-command deploy script

It's free. No email gate. Just value.
